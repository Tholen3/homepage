    Steps to push to the repo
  • Open Github. Click "Sync" to make sure everything is up-to-date.
  • Navigate to Documents/Github/homepage and right-click the Index.html file. Select "Edit with Atom."
  • Make any updates you desire to the page.
  • BE SURE TO SAVE!
  • Open Github and go to the "Changes" tab at the top.
  • Type in the title of your commit and a brief description. Click "Commit to GH."
  • Go back to the "History" tab at the top and Sync to make sure it's synced on GitHub.
